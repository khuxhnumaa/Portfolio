# 🌐 Portfolio Website

A modern and responsive personal portfolio website showcasing my projects, skills, education, and experience.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Interactive and modern UI
* Hero section with introduction
* About Me section
* Education section
* Projects showcase
* Contact section
* Smooth animations and transitions

## 🛠️ Tech Stack

* JavaScript
* HTML5
* CSS3
* npm

## 📁 Project Structure

```text
src/
```

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/khuxhnumaa/Portfolio.git
```

### Navigate to the project directory

```bash
cd Portfolio
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm start
```

If your project uses a different command, check the scripts section in `package.json`.

## 📜 Available Scripts

The available npm scripts can be found in `package.json`.

Common commands:

```bash
npm start
npm run dev
npm run build
```
